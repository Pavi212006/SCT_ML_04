ma# SkillCraft ML Task 4: Hand Gesture Recognition

## 📌 Project Description
This project is part of my **SkillCraft Technology Machine Learning Internship**. It uses **Google’s MediaPipe AI** to recognize hand gestures in real-time. Instead of using a manual dataset, I implemented a pre-trained Google model that identifies 21 hand landmarks to classify gestures like Thumbs Up, Victory, and Open Palm via a webcam.

## ⚙️ Setup
To run this project, ensure you have **Python** installed.

1. **Clone the repository:**
   git clone https://github.com
   cd SCT_ML_04

Install required libraries:
bash
pip install opencv-python mediapipe

🚀 How to Run
Run the Python script using the following command:
python SCT_04.py

A window will open showing your webcam feed. It will automatically detect your hand landmarks and display the recognized gesture on the screen.
